# Fusion 复合学习与记忆 (Compound Learning)

> 目标: 让明天的 Agent 比今天的更聪明。

## 1. 核心思想

Agent 通过复盘自己的会话日志，提取模式、教训和用户偏好，并自动更新项目记忆文件（`MEMORY.md`）。

## 2. 提取维度

- **成功模式 (Patterns)**：哪些方法生效了？
- **雷区 (Gotchas)**：哪些操导致了报错或死循环？
- **用户偏好 (Preferences)**：Commander 喜欢简洁还是详尽？
- **核心决策 (Decisions)**：为什么选了方案 A 而非 B？

## 3. 操作流程

每当任务完成或会话结束，Agent 必须生成一个小的学习摘要，并将其存入项目 `memory/` 目录下的日期文件中。

## 4. 周期演化

每周或每完成一个重大里程碑，由 Lead 角色汇总所有日期文件，更新 `.claude/rules` 目录下的技能规则。
